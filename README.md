1. Создайте консольный проект
2. Определите целевой интерфейс (Target), который ожидает ваш основной код:
• По типу: IMessageSender с методом Send(string header, string body) или IPaymentService с методами Pay(decimal amount) и Refund(decimal amount выберите любой подходящий пример.
3. Реализуйте класс «Adaptee» (адаптируемый):
•
Это может быть сторонняя библиотека или имитация таковой (например, класс, который принимает совсем другой формат данных либо имеет методы с другими сигнатурами).
•
Важно показать, что его интерфейс не совпадает с целевым.
4. Создайте класс-адаптер (Adapter), который:
•
Реализует целевой интерфейс (Target).
•
Хранит внутри ссылку на Adaptee (или агрегирует несколько зависимостей).
• Переводит вызовы от Target к методам Adaptee, преобразуя (если нужно) типы и сигнатуры параметров.

Критерии оценки
•
Корректная реализация паттерна Adapter (есть Target, Adaptee, Adapter).
• •
